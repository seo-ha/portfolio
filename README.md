## 홈페이지 설명
pc화면을 모티브로 작업한 포트폴리오입니다.

심플하면서 익숙한 디자인을 했으며, Component의 재활용을 높혔습니다.

+ filter를 사용해 메인페이지의 원하는 태그를 찾을 수 있습니다.
+ 카테고리의 빈 곳을 잡고 드래그하면 움직이는 효과를 줬습니다.
+ Email.js를 사용해 메일을 저에게 보낼 수 있습니다.
